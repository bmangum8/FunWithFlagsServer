# Fun With Flags Server

## Prerequisites
1. Install JSON Server

```sh
npm i json-server
```



2. Client Side Repo

link to fun with flags repository


## Demo
1. Clone this repository and change to the directory in the terminal.

```sh
git clone git@github.com:bmangum8/FunWithFlags.git
```

2. Run the server.

